Find Pivot Index

We define the pivot index as the index where the sum of the numbers to the left of the index is equal to the sum of the numbers to the right of the index. If no such index exists, we should return -1. If there are multiple pivot indexes, return the left-most pivot index.

The pivot index is the index where the sum of all the numbers strictly to the left of the index is equal to the sum of all the numbers strictly to the index's right. Given an array of integers nums, calculate the pivot index of this array

This code calculates the total sum of the array elements and iterates through the array, keeping track of the left sum and calculating the right sum. If the left sum and right sum are equal at any index, it returns that index as the pivot index. If no such index is found, it returns -1
